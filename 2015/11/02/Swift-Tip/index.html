Swift-Tip

Swift开发必备tip

Swift刚发布的时候，我就实践用swift写了一个项目（是公司的真实项目），当时swift给我的感觉就是语法的改变，更加精炼，当时用swift也只是简单的将oc代码用swift代替一下，其实swift的真正强大之处是 新特性，最近在看“喵神”写的swifter_100个Swift开发必备Tip，真的不错，主要讲了一些新特性，我一边学习一边会记录笔记。这里的绝大多数例子都是书中的例子，正在学习swift的同学强烈推荐你们看一下。

柯里化（Currying）

  • 柯里化函数

    func addTowNumbers(a: Int)(num: Int) -> Int {
    return a + num
}

  • 只传入第一个参数得到需要传入剩余参数的新的方法

    let addTo4 = addTowNumbers(4)
// Int -> Int

let add6 = addTo4(num: 6)
// 10

let add7 = addTo4(num: 7)
// 11

@autoclosure

  • @autoclosure 该方法的参数是接受一个函数
func logIfTrue(predicate: () -> Bool) {
    if predicate() {
        print("is true")
    }
}
  • 不加 @autoclosure（{11 > 2}）闭包的简写
logIfTrue({11 > 2})
  • 添加 @autoclosure 关键字
func logIfTrue(@autoclosure predicate: () -> Bool) {
    if predicate() {
        print("is true")
    }
}
  • 加 @autoclosure（11 > 2）部分会自动识别成闭包 语法更加简练
logIfTrue(11 > 2)

??操作符

  • ??操作符可以快速地对nil进行条件判断

    var level: Int?
var startLevel = 1

var currentLevel = level ?? startLevel
// 因为level == nil 所以 currentLever = 1
