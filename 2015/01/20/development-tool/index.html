开发必备工具

日常开发常用工具

名称 描述
SourceTree 强大的图形页面git管理工具
iTerm2 强大的命令行工具，支持多开，比 terminal 强太多
Oh My Zsh zsh开源插件，扩展了很多功能，简化语法
SwitchHosts host图形页面管理工具非常便捷
Kaleidoscope 强大的文件diff工具，可以配置到 git 中，使用 git difftool调起
MacDown 离线 markdown 编辑器
Mark Man UI标注工具
LICEcap 快速生成 git 工具

iOS逆向相关

名称 描述 相关资料
class-dump 利用runtime机制，将存储在Mach-O文件中的头文件信息提取出来，并生成对应的.h文件。 使用方法 https://blog.csdn.net/vbirdbest/article/details/53520889
Hopper Disassembler（自己的百度网盘）（官网） 超级强大的反编译软件，不仅可以把机器码解析成汇编，还能解析出相似与Objc的伪代码。
