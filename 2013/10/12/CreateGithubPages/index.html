创建Github静态网站

简单几步创建Github个人网站

1.登录Gitbub（没账号的注册一个）点击 + New repository，新建一个repository

2.Repository name 需要命名为:（你的github用户名).github.io

3.点击 Create repository 之后就创建好了一个 repository

4.在页面右边点击 Settings

5.滑到下面点击 Launch automatic page generator

6.下面这个页面就是网页的编辑页面，可以设置title、body等等，内容编辑使用Markdown方式。

7.编辑页面后点击页面下方的 Continue layouts

8.到这一步其实一个静态的页面已经创建成功，在最上面是可选的几个主题模板，点击：Publish page 发布页面。

9.这时用之前的域名http://username.github.io就可以访问页面了。是不是超级简单！！！

end!
