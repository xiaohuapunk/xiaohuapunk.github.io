OAuth2.0

概念

OAuth2.0 是一种授权协议，现在很多App或者一些网站需要注册登录后才可以使用，注册过程又比较繁琐有些用户并不想注册太多账号，从而会造成一部分用户流失，为了简化这个过程可以使用第三方登录的功能，比如QQ、微信、微博等都提供授权登录功能，这个授权登录所应用的技术就是OAuth2.0；

引用一张网络图（豆瓣网站使用QQ授权登录的 流程图）

关键步骤分析

1.客户端点击QQ登录，服务器将请求重定向到QQ的授权服务器

2.向QQ服务器发送请求，QQ服务器返回 authentication code；

3.客户端通过callback将code传给自己服务器；

4.自己服务器使用code向QQ授权服务器请求 token；到这一步OAuth的认证过程已经完成

5.有了 token 就可以获取用户QQ的用户信息；

简单来说就是先获取 code，然后使用code在获取token，有了token就可以访问开放的接口，比如用户基本信息等；

  • code：为了安全考虑，code通常都是唯一性，且只能使用一次；
  • token：通常都会有expires超时时间，有些系统还会设计一个refresh_token，用于刷新 token 的超时时间；
